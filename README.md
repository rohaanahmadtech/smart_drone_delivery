# 🚁 Smart Drone Delivery System
### A Real-World Pathfinding & Graph Algorithm Optimization System

> **Algorithms Project | Type:** Delivery Optimization | **Status:** ✅ Complete | **Made With:** Python 🐍

---

## ✨ Overview

**Smart Drone Delivery System** is an intelligent navigation solution that applies powerful Graph Algorithms to find the shortest and safest flying routes for drone deliveries in urban environments.

The system simulates real-world delivery challenges through a comprehensive drone network featuring:

- 🏙️ Multiple Delivery Points
- 🚫 No-Fly Zones
- 🌤️ Weather-Aware Routing
- ⚡ Battery-Optimized Paths
- 🔋 Charging Stations
- 🗺️ Interactive Visualizations

This project transforms complex algorithmic concepts into a practical, real-world application with stunning visual feedback.

---

## 🚀 Key Features

- ✅ Shortest Path Detection (Dijkstra)

## 🧠 Algorithms Implemented

| Algorithm | Purpose |
|-----------|---------|
| **Dijkstra** | Finds shortest & safest flying routes |
| **Minimum Spanning Tree (MST)** | Optimizes delivery network connections |
| **BFS (Breadth First Search)** | Explores area level-by-level |
| **DFS (Depth First Search)** | Deep exploration of delivery zones |

---

## 📊 Algorithm Working

### Dijkstra's Algorithm →
Calculates optimal path considering:
- Distance between points
- Weather conditions (rain, wind)
- No-fly zone avoidance
- Battery consumption rates

### Minimum Spanning Tree →
Builds efficient delivery network by:
- Connecting all delivery points
- Minimizing total path cost
- Identifying critical routes

### DFS/BFS →Enables:
- Complete area coverage
- Backup route discovery
- Zone exploration patterns

## ▶️ Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/Smart-Drone-Delivery-System.git
```

### 2. Navigate into the Project Directory
```bash
cd Smart-Drone-Delivery-System
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Run the System
```bash
python main.py
```
