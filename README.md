# 🚁 Smart Drone Delivery System

A comprehensive drone delivery route optimization system using graph algorithms.

## 📋 Features

- **Dijkstra's Algorithm**: Find shortest and safest flying routes
- **Minimum Spanning Tree (MST)**: Optimize delivery network
- **DFS/BFS**: Area exploration and search algorithms
- **Battery Management**: Automatic charging station routing
- **Weather Integration**: Dynamic route costs based on weather
- **No-Fly Zones**: Obstacle avoidance

## 🛠️ Technologies

- Python 3.8+
- NetworkX - Graph algorithms
- Matplotlib - 2D visualizations
- Folium - Interactive maps
